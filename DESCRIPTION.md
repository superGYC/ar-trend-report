# AR Trend Report - 应收账款趋势分析报告生成器

一键生成专业的应收账款趋势分析报告，包含环比分析、年度趋势可视化及四图表仪表盘。

## 核心功能

📈 **趋势分析** — 自动识别12月+应收余额走势，标注最高/最低点  
📊 **环比监控** — 计算月度环比变化率，绿涨红跌一目了然  
💰 **回款对比** — 新增应收 vs 实际回款双柱对比图  
⚠️ **逾期预警** — 逾期金额趋势跟踪，20%警戒线智能提示  
📝 **执行摘要** — 自动生成关键指标、洞察发现及改进建议

## 适用场景

- 财务月度/季度汇报
- 管理层应收健康度检查
- 销售回款效率分析
- 现金流预测辅助

## 输入输出

**输入**: CSV/Excel (支持中英文列名)  
**输出**: PNG四图表仪表盘 + Markdown执行报告

## 快速开始

```bash
skillhub install superGYC/ar-trend-report
python scripts/generate_ar_report.py --input data.csv --output report
```

## 特点

✅ 零配置即用 — 内置12个月测试数据  
✅ 智能列名识别 — 自动适配中英文表头  
✅ 专业可视化 — 基于 Matplotlib 的高质量图表  
✅ 可扩展 — 支持自定义报告标题和输出格式
